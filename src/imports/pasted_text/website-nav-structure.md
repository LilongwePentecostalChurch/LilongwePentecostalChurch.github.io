**COMPLETE WEBSITE NAVIGATION STRUCTURE**

Update the entire website navigation to reflect the following complete structure. The navigation bar is fixed to the top of every single page across the entire website. Background: #1A0500 dark maroon semi-transparent with backdrop blur. Left: LPC circular badge logo 40px height. Center: all main nav tabs. Right: TikTok, YouTube, Facebook, Instagram icons in small orange circles 32px each, then orange pill CTA button "Join Us Sunday" in Montserrat Bold white. Thin orange bottom border on nav bar.

Every nav tab with a dropdown has a small orange downward chevron arrow beside the label. On hover the chevron rotates 180 degrees smoothly as the dropdown opens. Dropdown opens with smooth fade-in slide-down animation 0.3s. Dropdown closes when mouse leaves both the tab and the dropdown panel.

On mobile: hamburger menu icon in orange top right. Tapping opens full screen dark maroon overlay. All main tabs listed vertically in TAN-BUSTER Bold white large text. Each tab with a dropdown shows a plus icon on the right. Tapping the plus expands the sub-pages list below that tab with smooth accordion animation. Social icons and Join Us Sunday button at the bottom of the mobile menu.

**ACTIVE PAGE INDICATOR**
Whichever page the user is currently on — that nav tab label turns orange and gets a thin orange underline beneath it. The corresponding dropdown sub-page row also gets a thin orange left border and slightly brighter text to show the active page within the dropdown.

---

**BREADCRUMB NAVIGATION**
Every sub-page has a breadcrumb trail below the hero section in Montserrat small white. Format: Home › Tab Name › Page Name. Each breadcrumb link turns orange on hover. The current page name is white and not clickable. Example: Home › About › Our Story

---

**SCROLL BEHAVIOUR**
The navigation bar is always fixed to the top of the screen. As the user scrolls down on any page the nav background increases in opacity from semi-transparent to fully solid #1A0500 — smooth transition over the first 80px of scroll. The LPC logo and all nav elements remain fully visible at all scroll depths.

---

**LIGHT AND DARK MODE**
Dark mode default: nav background #1A0500, all labels white, orange hover, orange CTA button, orange bottom border.
Light mode: nav background warm white #FFF8F0 with orange bottom border, all labels dark maroon #1A0500, orange on hover, CTA button stays orange fill white text, dropdown background switches to warm white with orange border and dark maroon text, featured cards switch to warm white background with dark maroon text and orange accents.

---

**GLOBAL RULES**
This navigation structure applies to every single page on the entire LPC website without exception. Every page — homepage, all about pages, all sermon pages, all ministry pages, all events pages, all connect pages, all children pages, all community pages, gallery page, all give pages, all leadership pages — uses this exact same navigation bar at the top. The floating social media side panel — TikTok, YouTube, Facebook, Instagram — is present on every page. The standard LPC footer is present on every page. Light and dark mode toggle is present on every page. All navigation links are designed as clean editable blocks so URLs and labels can be updated through the CMS after deployment to Vercel without touching code.
